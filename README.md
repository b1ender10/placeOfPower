# Happiness Tracker

Приложение для трекинга личного ощущения счастья по часам.

## Запуск

```bash
npm install
npm run dev
```

## Функции

- **День** — график за выбранный день, добавление записей (дата, час, рейтинг 1–10, комментарий, категория)
- **Статистика** — неделя / месяц / всё время, средний рейтинг, графики, разбивка по категориям
- **Категории** — добавление, редактирование, удаление (если нет записей)

Данные хранятся в localStorage.
