# Место силы

Приложение для мягкого отслеживания состояния, мыслей и действий (ранее в репозитории — happiness tracker).

## Запуск

```bash
npm install
npm run dev
```

Сервер разработки: порт **5174** (`vite.config.ts`).

## Данные

- **Продакшен и статическая сборка** (`npm run build` / любой хостинг без бэкенда): снимок хранится в **`localStorage`** (ключ `happy_app_snapshot_v1` в `src/lib/appData.ts`).
- **Dev**: при первом запуске, если в `localStorage` пусто, данные **один раз** подтягиваются из `GET /api/data` (файл `src/data/app-data.json`), записываются в `localStorage` — так сохраняются текущие локальные данные разработчика.
- **Dev — зеркало и бэкапы**: при каждом сохранении дополнительно выполняется `PUT /api/data` — плагин `vite-plugin-happy-data.ts` пишет файл, ежедневный бэкап в `src/data/backups/`, защита от «пустого» снимка.

Импорт стартового JSON из `src/data.js`: `npm run data:import`.
